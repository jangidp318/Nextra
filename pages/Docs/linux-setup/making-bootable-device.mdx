# Making the bootable Device

## Requirements




1. 8GB or above USB drive
   - NOTE: Rufus will remove everything from your USB device. If you have any important data, please move it somewhere safe before you go through this process.
2. Latest version of Rufus
   - You can find the latest verision of Rufus [here](https://rufus.ie/en/)
3. Ubuntu ISO image.
   - You can find the latest verision of Ubuntu ISO image [here](https://ubuntu.com/#download)

## Set-up

#### Step 1: - Start the Rufus tool to create a bootable flash device.

#### Step 2: - Select the USB device to boot.
   
   <img src="https://ucarecdn.com/06389f8d-370d-4dca-9911-9f843a7a2222/image.jpg" style={{ margin: "50px" }}/>

#### Step 3: - Now Select Patition schema as MBR

   <img src="https://ucarecdn.com/953d6e08-269c-4976-859c-bbe682bcf51d/image-3.png" style={{ margin: "50px" }} />

#### Step 4: - Select the File system as FAT32

   <img src="https://ucarecdn.com/d6fe57e9-36e7-4cef-ba9f-ffc099a45afd/image-4.png" style={{ margin: "50px" }} />


#### Step 5: - Select the downloaded ISO file as shown in the below image.

   <img src="https://ucarecdn.com/017934be-bbd9-4181-a0ea-3bfe3937efe7/image-1.png" style={{ margin: "50px" }} />

#### Step 6: - Now click on Start.

   <img src="https://ucarecdn.com/a959f066-996b-492d-996d-206b461a8557/image-5.png" style={{ margin: "50px" }} />

#### Step 7: - You should see the following window. Now click on Yes.

   <img src="https://ucarecdn.com/9ee16ca2-66f6-4716-adda-09761e64f52d/image-6.png" style={{ margin: "50px" }} />

#### Step 8: - You should see the following window. Leave the defaults and click on OK.

   <img src="https://ucarecdn.com/9ee16ca2-66f6-4716-adda-09761e64f52d/image-6.png" style={{ margin: "50px" }} />

#### Step 9: - Now you'll get a warning. If you have backed-up your data from USB select OK or else backup the data and come again.

   <img src="https://ucarecdn.com/280b9740-da35-4703-b106-1eb9f2b72efb/image-7.png" style={{ margin: "50px" }} />

#### Step 10: - Now leave the setup to get done.
   <img src="https://ucarecdn.com/f62e89ac-c1b3-405d-9fef-bfd7959b4c78/image-8.png" style={{ margin: "50px" }} />

   <img src="https://ucarecdn.com/40488fba-cca3-49fa-99c5-79f3de6c4af6/image-15.png" style={{ margin: "50px" }} />

#### Hurray...! You made the bootable flash device ; )
